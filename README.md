# 📚 Nexus DocuChat AI (Hybrid RAG)

An Enterprise-Grade Document Intelligence Platform powered by Generative AI and Hybrid Retrieval-Augmented Generation (RAG) architecture. This application allows users to converse with their personal documents (PDF, DOCX, TXT) seamlessly.

## 🚀 Key Features
- **Omni-Reader:** Upload and extract text from PDFs, Microsoft Word (.docx), and plain text files.
- **Hybrid RAG Architecture:** Utilizes `HuggingFace` models for local, privacy-first vector embeddings, and `Google Gemini 1.5 Pro` for high-level language generation.
- **Lightning Fast Vector Search:** Powered by Meta's `FAISS` (Facebook AI Similarity Search) database.
- **Chat Export:** One-click download of your entire AI consultation history for offline viewing.

## 🛠️ Tech Stack
Python | Streamlit | LangChain | HuggingFace (Sentence Transformers) | FAISS | Google Gemini API
